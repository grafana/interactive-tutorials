Adding Testing & Synthetics

This tutorial will guide you through adding Testing & Synthetics using Grafana's plugin system.

Section 1: Accessing Plugins

In this section, you'll navigate to the Plugins section in Grafana to manage and install new plugins.

  • Open the main menu and click Administration
  • Click Plugins and data
  • Click Plugins

Section 2: Installing Synthetic Monitoring

Now, you will search for and install the Synthetic Monitoring plugin.

  • Type "Synthetic Monitoring" in the search bar
  • Select Synthetic Monitoring Plugin from the search results
  • Click Install to add the plugin
