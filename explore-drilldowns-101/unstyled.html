Explore Drilldowns

In this part, you'll explore the Drilldowns experience to pivot from metrics to related metrics and logs.

Section: Drilldown Metrics

  • Select Metrics from the left navigation menu.
  • Notice the Data source dropdown at the top. This shows your currently selected Prometheus data source. Click it to switch between different data sources if you have multiple configured.
  • Optional: Use the filter field to narrow down metrics by label values (e.g., job = "my-service" or instance = "localhost").
  • Look at the first metric card. Each metric card shows:
    • The metric name as a title
    • A preview chart showing recent data trends
    • A Select button to choose the metric for drilldown
  • Click the Select button on any metric to choose it for drilldown exploration.
  • Open the Related metrics tab.
  • Open the Related logs tab.
