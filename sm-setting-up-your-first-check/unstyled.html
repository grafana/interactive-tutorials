Setting up our first check in Synthetic Monitoring

In this tutorial, you'll learn how to create your first synthetic monitoring check to proactively monitor your applications and services. Synthetic Monitoring allows you to simulate user interactions and monitor availability from multiple locations around the world.

By the end of this tutorial, you'll know how to:

  • Navigate to Synthetic Monitoring in Grafana
  • Create a new check
  • Configure check settings and probes
  • View check results and metrics
