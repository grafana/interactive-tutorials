Namespace predictions

  • Predictions require more than an hour of data to be collected. Click the time range picker to adjust the window.
  • Select Last 24 hours so predictions have enough history to work with.
  • Click Namespaces in the main menu to view the list of namespaces in your environment.

  • Select the first namespace in the list to open its detail view.
  • Open the Namespace CPU prediction modal to see the forecasted trend.
  • Close the prediction modal when you are done reviewing it.
  • Open the Namespace Memory prediction modal to compare the memory forecast.
  • Close the modal to return to the dashboard.
