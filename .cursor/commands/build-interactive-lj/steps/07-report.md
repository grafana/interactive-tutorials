# Step 7: Report and Next Steps

Summarize the session and provide guidance for creating a PR.

---

## Summary

Display a completion summary showing: total milestones, how many are fully interactive vs partial, a list of all created content.json files, and next steps (review, stage, commit, PR).

---

## PR Description

When the user asks for a PR description, provide it in a markdown code block so they can copy it. Include: a summary of what was added, a table of milestones with file paths and block counts, key interactive features, and testing status.
