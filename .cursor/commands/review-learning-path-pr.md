## Command: /review-learning-path-pr

Review a learning path pull request in `grafana/interactive-tutorials`. Guide the reviewer **phase by phase** — explain what you're doing, stop at each checkpoint, and wait for their reply before continuing.

When invoked, read and follow **[review-learning-path/SKILL.md](../skills/review-learning-path/SKILL.md)** in order. Do not skip phases or checkpoints.

---

## First message to the reviewer

> I'll walk you through reviewing this learning path PR in order: checkout → static audit → path consistency → live testing (Playwright, then Pathfinder) → GitHub comments → submit.
>
> **Phase 0:** Share the PR — a GitHub URL or number for `grafana/interactive-tutorials`.

Do not start Phase 1 until the reviewer shares the PR and you have checked out the branch.
